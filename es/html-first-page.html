Esercizio 1: Creare la Prima Pagina

Base 30 min

Istruzioni

Obiettivo

Imparare la struttura base di un documento HTML e creare la tua prima pagina web.

Compiti

  1. Crea un nuovo file HTML chiamato index.html
  2. Aggiungi la struttura base del documento HTML5 con dichiarazione DOCTYPE
  3. Includi i tag appropriati <html>, <head> e <body>
  4. Imposta il titolo della pagina a "La Mia Prima Pagina Web"
  5. Aggiungi un'intestazione principale (h1) con il tuo nome
  6. Aggiungi un paragrafo per presentarti
  7. Assicurati che sia impostata la codifica caratteri corretta (UTF-8)
  8. Aggiungi un meta tag viewport per la responsività mobile

Output Atteso

Una pagina HTML valida che mostra il tuo nome come intestazione principale e un breve paragrafo di presentazione sotto.

Suggerimenti

  • Usa un'indentazione appropriata per rendere il codice leggibile
  • Valida il tuo HTML usando il W3C Validator
  • Testa la tua pagina in un browser web

Soluzione

Clicca per rivelare la soluzione

Nota: Prova a completare l'esercizio da solo prima di visualizzare la soluzione.

Codice HTML:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>La Mia Prima Pagina Web</title>
</head>
<body>
    <h1>Mario Rossi</h1>
    <p>Ciao! Sono Mario Rossi e questa è la mia prima pagina web. 
       Sto imparando HTML per diventare uno sviluppatore web.</p>
</body>
</html>

Spiegazione:

  • <!DOCTYPE html> dichiara questo come documento HTML5
  • <html lang="it"> specifica la lingua del documento come italiano
  • <meta charset="UTF-8"> imposta la codifica caratteri
  • <meta name="viewport"...> rende la pagina responsive
  • <title> imposta il titolo della pagina mostrato nella scheda del browser
  • <h1> crea l'intestazione principale
  • <p> crea un paragrafo
